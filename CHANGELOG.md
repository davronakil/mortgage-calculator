# Changelog

All notable changes to this project are documented in this file.

## [0.2.0] - 2026-04-16

### Added

- Multi-tab planning suite across Residential, Commercial, and Other calculators.
- County presets for Dallas-area counties and richer mortgage planning outputs.
- Saved preset workflow with export/import support.
- Amount-or-percent smart input modes for down payment and concession analysis.
- Amortization visualization, payoff schedule recommendations, and target income planner.
- Loan-program presets (Conventional, FHA, VA, Physician, Jumbo).
- Footer credit linking to Akil Real Estate.

### Changed

- Product renamed to `Dallas Real Estate Calculator`.
- Expanded documentation set for architecture, development, assumptions, and operations.

## [0.1.0] - 2026-04-16

### Added

- Initial Next.js mortgage calculator setup.
